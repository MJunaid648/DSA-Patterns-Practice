# 📌 Core Patterns Covered

## 1. Hashing (Frequency / Map / Set)
Used for:
- Frequency counting
- Anagrams
- Duplicate detection
- Lookup optimization

Key idea:
> Store information as you iterate and reuse it instead of recomputing.

---

## 2. Two Pointers
Used for:
- Sorted array problems
- Pair finding (Two Sum II)
- Palindromes
- Container problems
- Removing duplicates

Key idea:
> Use two indices moving inward or forward to reduce nested loops.

---

## 3. Sliding Window
Used for:
- Subarray/substring problems
- Fixed window size problems
- Variable condition-based window problems

Key idea:
> Expand → validate → shrink → update answer

---

## 4. Kadane’s Algorithm
Used for:
- Maximum subarray problems
- Maximum sum / profit problems

Key idea:
> At each index, decide:
- extend previous subarray
- or restart from current element
