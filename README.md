# 📌 Core Patterns Covered

## 1. Hashing (Frequency / Map / Set)

Used for:

- Frequency counting
- Anagrams
- Duplicate detection
- Lookup optimization

Key idea:

> Store information while iterating and reuse it instead of recomputing.

Mental trigger:

> "Have I seen this before?"

---

## 2. Two Pointers

Used for:

- Sorted array problems
- Pair finding (Two Sum II)
- Palindromes
- Container problems
- Removing duplicates (in-place operations)

Key idea:

> Use two indices moving inward or forward to eliminate unnecessary comparisons.

Mental trigger:

> "Can I reduce O(n²) to O(n) by moving two pointers instead of nesting loops?"

---

## 3. Sliding Window

Used for:

- Subarray / substring problems
- Fixed window size problems
- Variable condition-based window problems

Key idea:

> Expand → maintain condition → shrink → update result

Mental trigger:

> "Is this a contiguous range problem?"

---

## 4. Kadane’s Algorithm

Used for:

- Maximum subarray problems
- Maximum sum / profit problems

Key idea:

> At each index, decide:

- extend previous subarray
- or restart from current element

Mental trigger:

> "Do I need best result ending at every index?"
